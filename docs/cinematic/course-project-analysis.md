## 4. 電話 CTA までの導線

- 電話番号への接点は 3 箇所: ①notes 内の番号（**プレーンテキスト・タップ不可**） ②CTA の朱ボタン（tel:） ③CTA の番号リンク（tel:）
- PC では Hero → 空ギャラリー（pin 1 ステージ）→ ボード → CTA と、**相談したい人が空の額縁を 1 画面分通過させられる**のが最大の摩擦
- モバイルでは notes の番号がタップできないため、「相談ください」と読んだ直後の行動が 1 スクロール先の CTA まで遅延する [推定: 改善候補]
