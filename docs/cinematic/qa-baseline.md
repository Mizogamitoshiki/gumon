## スクラブ動画の内容確認（A3条件の履行）

`cuisine-cinematic-opt.mp4` = **10.0秒・1280×720**。実シーク確認の結果:

| 時間 | 内容 |
| --- | --- |
| 0–2s | 木卓に並ぶ完成料理の俯瞰（麻婆豆腐中央・小籠包/焼売の蒸籠・青菜・春巻・炒飯・海老等） |
| 2–4s | 麻婆豆腐の寄り＋小籠包の蒸籠（照り・湯気感） |
| 4–6.5s | 炒飯の寄り（照り・葱） |
| 6.5–8.5s | 海老マヨの皿の寄り（盛り付けの美） |
| 8.5–10s | 俯瞰の全景に戻る |

**判定**: 調理・炎・仕込みのカットは**存在しない**。映像は「完成した一皿＝答えの連なり」であり、Scene名「火の返事」（/about『麻婆豆腐の痺れも、火の返事です』＝仕上がりが返事）とは整合する。**章句の扱いは次のとおり調整**（A3の条件履行）:

- 章句は「映像のキャプション」ではなく「台所の三つの問い」として重ねる（問いのテキスト × 答えの映像、という問答構造）
- 「火に問う」は麻婆豆腐の寄り（v≈2.8s）に重なるよう配時 — 唯一の完全一致点
- 「素材に問う」は俯瞰の全景（v≈0.9s）、「時間に問う」は炒飯の寄り（v≈4.7s）に配時。**「時間に問う」と映像の一致は弱い** — G2レビューで判定を仰ぐ（不一致と判断されれば章句3を削除する。Motion表の削除条件どおり）
- 映像のトーンは「明るめの木卓・自然光」で、imagery方針（暗背景・一点光の陰影）より明るい。darken/フィルタで馴染ませている現行処理を維持しつつ、G2で許容可否を判定
